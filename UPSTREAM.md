## Метод перевода

Английский upstream → русский (пакетный перевод через Claude Sonnet). Сохраняет тон и структуру оригинала, с естественными для русского технического читателя выражениями.

- Устоявшиеся английские термины (TensorFlow / PyTorch / RAG / MLOps / LLM / API / OAuth / REST и т.д.): сохранены на английском
- Блоки кода, пути, URL, команды: без изменений
- Тело / заголовки / таблицы / списки: переведены на естественный русский
- Frontmatter `name:`, `description:`, `vibe:`: переведены; `color:`, `emoji:`: сохранены

## Политика синхронизации

- Отслеживание ветки `main` upstream
- Новые агенты upstream будут переводиться постепенно
- Крупные структурные изменения (переименование директорий и т.п.) — в течение 1 недели
- Версия upstream обновляется в этом файле при каждой синхронизации
